# TradeQuest

**TradeQuest** — образовательная RPG-игра про инвестиции и трейдинг. Игрок регистрируется, подтверждает 18+, покупает и продаёт виртуальные акции/криптовалюты, смотрит свечные графики, выполняет квесты и соревнуется с другими участниками рейтинга.

Денисов Арсений Андреевич (Группа: ИД-23.1 Б1-24)

## Что есть в проекте

- графическое окно на Python Tkinter;
- простая регистрация по имени;
- обязательная кнопка подтверждения 18+;
- виртуальный баланс и портфель;
- акции и криптовалюты;
- свечные графики с объёмом и скользящей средней;
- попытка загрузить реальные данные через Yahoo Finance chart endpoint;
- если реальные данные недоступны, включается встроенная симуляция рынка;
- графики обновляются только на вкладке **Рынок**, чтобы не было лишнего мерцания;
- боты, замаскированные под обычных игроков;
- боты торгуют, получают XP и двигаются в рейтинге;
- расширенная система квестов;
- учебные мини-вопросы;
- сохранение прогресса в SQLite.


## Зависимости

Дополнительные библиотеки устанавливать не нужно. Используются только стандартные библиотеки Python:

- tkinter
- sqlite3
- urllib
- json
- random
- threading
- datetime

## Важно

Это учебный проект. В игре используются виртуальные деньги, а сама программа не является инвестиционной рекомендацией.
